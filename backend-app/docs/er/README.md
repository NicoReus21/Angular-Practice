# Diagramas Entidad-Relación (ER)

Este directorio contiene los diagramas ER de los módulos clave del sistema en formato PlantUML (`.puml`).

- Material Mayor: `docs/er/material-mayor.puml`
- Bombero Accidentado: `docs/er/bombero-accidentado.puml`
