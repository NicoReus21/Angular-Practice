# Perfil de Usuario

**Nombre:** Capitán  
**Descripción:** Lidera la compañía, valida información operativa y aprueba documentación clave.  
**Responsabilidades:**  
- Revisar y aprobar reportes de accidentes y documentos asociados.

- Autorizar envíos o escalaciones cuando corresponda.  
**Accesos clave:**  
- Flujos de Bombero Accidentado y seguimiento.  
- Consulta y carga de documentos requeridos.
