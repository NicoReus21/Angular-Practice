# Perfil de Usuario

**Nombre:** Administrativo  
**Descripción:** Gestiona la operación administrativa diaria, ingreso y actualización de documentos y seguimiento de plazos.  
**Responsabilidades:**  
- Crear y mantener registros de procesos y documentos.  
- Monitorear plazos y coordinar notificaciones o recordatorios.  
- Validar consistencia de datos antes de su aprobación.  
**Accesos clave:**  
- CRUD de usuarios y procesos que le competen.  
- Subida y validación de documentos.  
- Paneles de seguimiento y reportes básicos.
